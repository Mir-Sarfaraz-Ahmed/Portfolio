IoT-Based Smart Traffic Management System

Built using Arduino Uno, IR sensors, RF communication, and a gas sensor. The system dynamically regulates traffic based on vehicle density, prioritizes emergency vehicles, and detects potential fire hazards.

Key Features:

  • IR-based traffic density detection
  • Dynamic signal timing adjustment
  • RF-based ambulance priority override
  • Gas sensor fire alert mechanism
  • Real moving vehicle simulation
