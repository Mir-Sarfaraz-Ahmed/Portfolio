IoT-Based Smart Traffic Management System

An Arduino Uno powered intelligent traffic control model using IR sensors for density detection, RF-based ambulance priority override, and gas sensor-based fire detection system.

Key Features

  • IR-based traffic density detection
  • Dynamic signal timing adjustment
  • RF-based emergency vehicle override
  • Gas sensor fire alert mechanism
  • Real moving vehicle simulation

My Role in the Project

  • Designed and built the mechanical traffic model
  • Integrated IR sensors with Arduino Uno
  • Implemented RF-based emergency vehicle prioritization
  • Developed gas sensor fire detection module
  • Collaborated on presentation and documentation

Live Demonstration

Demonstration of RF-based emergency vehicle override system. The traffic light automatically switches to green for the approaching ambulance.
